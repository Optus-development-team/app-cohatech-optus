# app-cohatech-optus

Base frontend project scaffolded with Next.js, TypeScript, and pnpm.

## Structure

- `src/app` for the application shell and routes.
- `src/components` for reusable UI pieces.
- `src/features` for feature-focused modules.
- `src/services` for API clients and external integrations.
- `src/lib` for shared utilities and framework helpers.
- `src/styles` for global design tokens and styles.
- `src/types` for shared TypeScript types.

## Scripts

- `pnpm dev` starts the development server.
- `pnpm build` creates a production build.
- `pnpm start` runs the production server.
- `pnpm lint` runs linting.
- `pnpm typecheck` validates TypeScript types.
